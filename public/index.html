Request Header Parser Microservice


This website if for the Free Code Camp Back End API Request Header Parser Microservice project . It exposes a single enpoint at /api/whoami which returns infomation about your IP address, language, and the software you are using, in a JSON format.


By Abose Ukhun | Project Github

User stories:
  1. I can get the IP address, language and operating system for my browser.

Usage

The endpoint to the api can be found here:

https://bexis-request-header-parser.herokuapp.com/api/whoami

Example output:

  • Raw
{
        "software": "Windows XP 10.1; Win64; x64",
        "language": "en_EN",
        "ipaddress": "96.266.182.22"
}
